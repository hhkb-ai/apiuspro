间距与布局

遵循 Tailwind 4px 基准间距。移动端主要使用 px-4，桌面端使用 px-6/px-8 和 max-width 容器。

基础单位

1 step = 0.25rem / 4px

所有内边距、外边距和 gap 基本围绕 Tailwind spacing scale。

页面横向内边距

mobile px-4 · desktop px-6 / px-8

移动端保证内容不贴边，桌面端保持文档型密度。

卡片与区块

p-4 / p-5 / p-6 · py-8 / py-12

列表卡片比营销页更克制，适合高频扫描。

容器宽度

max-w-5xl / max-w-6xl / max-w-7xl / max-w-[1200px]

详情页偏窄，首页和列表页可更宽。

边框、圆角与阴影

结构主要靠 1px 边框建立层级，阴影使用克制；圆角基准为 8px。

--radius

0.5rem / 8px

派生 sm=4px、md=6px、lg=8px、xl=12px。
