## 一、更新网站内容

### 1. 本地推送代码到 GitHub

```bash
cd d:/projects
git add <修改的文件>
git commit -m "更新说明"
git push origin main
```

### 2. 修改容器启动命令为临时模式

1Panel → 容器 → apiuspro → 编辑 → 启动命令改为：
```
sleep infinity
```
保存并重启容器。

### 3. 进入容器终端拉取代码并构建

1Panel → 容器 → apiuspro → 终端，依次执行：

```bash
cd /app

# 清理旧代码（保留 node_modules）
find . -maxdepth 1 ! -name node_modules ! -name . -exec rm -rf {} +

# 克隆最新代码
git clone https://github.com/hhkb-ai/apiuspro.git /tmp/repo
cp -r /tmp/repo/* /app/
rm -rf /tmp/repo

# 安装依赖 + 构建
NODE_ENV=production npm run build

# 打包自定义服务器
npx tsup src/server.ts --format cjs --platform node --target node20 --outDir dist --no-splitting --no-minify
```

### 4. 制作新镜像

构建成功后，1Panel → 容器 → apiuspro → 制作镜像：
- 名称：`apiuspro`
- 标签：`latest`
- 确认（会覆盖旧镜像）

### 5. 改回启动命令并重启

1Panel → 容器 → apiuspro → 编辑：
- 启动命令：`bash -c "COZE_PROJECT_ENV=PROD node dist/server.js"`
- 保存并重启

## 二、SSL 证书管理

证书位置（OpenResty 容器内）：
- 证书：`/usr/local/openresty/nginx/ssl/apiuspro/apiuspro.cn.pem`
- 私钥：`/usr/local/openresty/nginx/ssl/apiuspro/apiuspro.cn.key`

### 证书到期续签（每 3 个月）

在阿里云 SSL 证书控制台申请新证书 → 下载 Nginx 格式 → 更新容器内文件：

1Panel → OpenResty 容器 → 终端：

```bash
cat > /usr/local/openresty/nginx/ssl/apiuspro/apiuspro.cn.pem << 'CERTEOF'
<粘贴新证书内容>
CERTEOF

cat > /usr/local/openresty/nginx/ssl/apiuspro/apiuspro.cn.key << 'KEYEOF'
<粘贴新私钥内容>
KEYEOF

nginx -s reload
```

## 三、常见故障排查

### 网站 502

Node.js 容器可能崩溃，检查：
1. 1Panel → 容器 → apiuspro → 查看日志
2. 确认启动命令为：`bash -c "COZE_PROJECT_ENV=PROD node dist/server.js"`
3. 容器重启后会自动构建（约 1-2 分钟）

### 网站 404

nginx 配置可能被覆盖，在 OpenResty 终端执行：

```bash
cat > /usr/local/openresty/nginx/conf/conf.d/apiuspro.cn.conf << 'NEOF'
server {
    listen 80;
    server_name apiuspro.cn www.apiuspro.cn;

    location ^~ /.well-known/acme-challenge/ {
        root /usr/local/openresty/nginx/html;
    }

    location / {
        proxy_pass http://127.0.0.1:5000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}

server {
    listen 443 ssl http2;
    server_name apiuspro.cn www.apiuspro.cn;

    ssl_certificate /usr/local/openresty/nginx/ssl/apiuspro/apiuspro.cn.pem;
    ssl_certificate_key /usr/local/openresty/nginx/ssl/apiuspro/apiuspro.cn.key;
    ssl_protocols TLSv1.2 TLSv1.3;
    ssl_ciphers HIGH:!aNULL:!MD5;

    location / {
        proxy_pass http://127.0.0.1:5000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
NEOF

nginx -s reload
```

### 容器一直重启

检查 OpenResty 容器是否运行 → 终端执行 `nginx -s reload`
