## 🔴 严重程度: 中等

### BUG-01: 第三方统计脚本加载失败
**严重程度**: 中等
**影响范围**: 全站
**复现步骤**:
1. 打开任意页面
2. 检查网络请求控制台

**实际结果**:
```
GET https://sdk.51.la/js-sdk-pro.min.js → 403 [FAILED: net::ERR_ABORTED]
```
该错误重复出现多次（至少7次），表明第三方统计SDK无法加载。

**预期结果**: 
- 51.la统计脚本应正常加载并返回200状态码
- 统计数据应能正常收集

**问题分析**:
- 可能原因：51.la账户未续费、API密钥失效、IP白名单限制
- 影响：无法追踪用户访问数据，影响运营分析

**修复建议**:
1. 检查51.la账户状态和API密钥
2. 确认SDK版本是否最新
3. 如无需该统计服务，建议移除相关代码，避免加载失败拖慢页面性能

## 🟡 严重程度: 低

### BUG-02: Next.js 路由过渡警告
**严重程度**: 低
**影响范围**: 所有使用路由跳转的页面

**控制台警告**:
```
Detected `scroll-behavior: smooth` on the `<html>` element.
To disable smooth scrolling during route transitions, 
add `data-scroll-behavior="smooth"` to your <html> element.
```

**问题分析**:
- Next.js App Router 中使用 smooth scroll 会产生警告
- 不影响功能，但会产生不必要的控制台噪音

**修复建议**:
在根布局文件 `src/app/layout.tsx` 的 `<html>` 标签添加：
```tsx
<html lang="zh-CN" data-scroll-behavior="smooth">
```

---

### BUG-03: 备案图标图片尺寸警告
**严重程度**: 低
**影响范围**: Footer 备案区域

**控制台警告**:
```
Image with src "/images/beian.png" has either width or height 
modified, but not the other. If you use CSS to change the size 
of your image, also include the styles 'width: "auto"' 
or 'height: "auto"' to maintain the aspect ratio.
```

**问题分析**:
- 图片只设置了宽度或高度，未同时设置另一维度
- 可能导致图片比例失真，影响可访问性

**修复建议**:
检查 `src/components/layout/BeianLinks.tsx` 中的图片样式，确保同时设置 `width` 和 `height` 或使用 `auto`：
```tsx
<Image 
  src="/images/beian.png" 
  width={16} 
  height={16} 
  alt="备案图标"
/>
```
